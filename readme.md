#### 上课时遇到的坑
1. 做 lab 时在 ok test 命令行之后添加 --local 选项即可设置本地测试
eg.
> python ok -q python-basics -u --local
2. 在对应项目文件夹下输入
> python scheme
来运行scheme解释器，输入
> python scheme -i <file.scm>
来交互执行Scheme程序
